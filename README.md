# Notifications

> Notification服务测试APP，可用来验证、测试后续对通知服务的修改，以及分析有Bug的通知消息。
  


**目前支持的功能：**

1. 覆盖了Android平台所有通知模板，以及自定义通知场景（支持自定义通知按钮）
2. 支持自定义通知提醒方式：悬浮、振动、响铃、呼吸灯、常驻
3. 支持自定义时间隔发送多条通知（进行压力测试）
4. 支持抓取当前通知栏所有活动通知的内容
